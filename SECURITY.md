# Security Policy

Please do not publish credentials, exported diagnostics containing personal information, or a working exploit in a public issue.

Report suspected vulnerabilities privately to the repository maintainers through GitHub's private vulnerability reporting feature. Include the affected version, macOS version, reproduction steps, expected impact, and whether the issue can expose traffic or proxy credentials.

The project does not intercept TLS and is not a VPN. A compromised local user account, upstream VPN, proxy server, Codex installation or operating system is outside its trust boundary.
